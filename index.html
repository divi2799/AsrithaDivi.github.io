Healthcare Pricing Transparency

I designed and implemented a complete end-to-end data pipeline and analytics solution using Azure and Databricks. The process began with extracting raw data via Azure Data Factory (ADF) and storing it in a secure container. Leveraging Databricks and the Medallion Architecture with Delta Live Tables, I transformed and cleansed the data before saving the processed version back into Azure Storage. The curated data was then loaded into a Microsoft SQL Serverless database, which I connected to Visual Studio Code for further development. To make the insights accessible, I built a web application using Flask APIs, enabling users to input treatment codes (such as CPT codes) and select their insurance plan from a dropdown. The application then delivers the best hospital options based on quality of care, optimal negotiation rates, and detailed treatment information, while also providing comparisons across different insurance plans.
